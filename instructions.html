Cookie Exercise

Notes:

  • You don't have to specifically match the UI in terms of font size, color, and measurements.
  • All elements (the text in the post, the post boxes, and the background, the drop down boxes) should be present in your recreation.
  • You need to use cookies to store the preferences. The keys you will be using are up to you. The number of cookies you will be using is up to you.
  • The number of HTMLs, JS files, etc are up to you.

Site to recreate

You will create a simple site that shows at least 2 posts of text. Somewhere in the page is a link to Set Preferences.

If you code, <a href="preferences">Set Prefs</a> this will connect to the helper with the path 
/preferences
.

When the user clicks on the Set Preferences, they will be directed to a page where they can set the

  • background color
  • post color
  • text color

For all three preferences, the user can set the colors to either

  • Red
  • Green
  • Blue
  • Yellow
  • Black
  • White

You can choose your own shade for the colors.

Let's say the user chose these set of preferences.

The user will see these colors in return.

These coloring scheme will remain for a day, even after the user closes the browser.

Test your work

  • Refresh the index page after setting the color preferences. This should still show the newly preferred colors.
  • Close your browser, and open the page again. The preferred colors should still be on display.

Improve the site

  • Set a default color palette from the start.
  • Allow the user to clear all cookies.
  • On the preferences page, set the drop down boxes to the current colors chosen. You'll need to use fancy JavaScript footwork here.
